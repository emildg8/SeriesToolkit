# Структура репозитория Script_Rename_ALLVideo

| Каталог | Назначение | Стабильная версия / ветка |
|---------|------------|---------------------------|
| `SeriesToolkit/` | Нормализация сериалов (GUI, движок, настройки) | **0.1.27** — текущая финальная линия; публикация `emildg8/SeriesToolkit` |
| `MediaInboxToolkit/` | Входная сортировка (inbox → Сериалы/Фильмы, TMDB, Blu-ray) | **0.1.0+**; разработка в ветке **`media-inbox-toolkit`** |
| `Fetch-VideoMetadata.ps1` | Общий модуль метаданных (корень репозитория) | Используют оба продукта |

Ветка **`media-inbox-toolkit`** содержит вынос inbox-логики из экспериментов в SeriesToolkit 0.2.x и не повышает версию SeriesToolkit.
